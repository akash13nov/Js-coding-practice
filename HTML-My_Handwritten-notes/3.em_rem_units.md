# 📏 EM vs REM Units in CSS

## 🧩 What Are They?
Both **`em`** and **`rem`** are **relative CSS units** used for scalable and responsive design.

| Unit | Full Form | Relative To |
|-------|-------------|-------------|
| `em` | Element-relative unit | Font size of the element’s **parent** |
| `rem` | Root element-relative unit | Font size of the **root (`<html>`)** element |

---

## 🧠 Why Use Relative Units Instead of `px`?
- **Accessibility:** Respects user’s browser font size settings.  
- **Responsive Design:** Adapts better across devices and screen sizes.  
- **Maintainability:** You can scale the entire UI by changing one value (usually the root font size).  
- **Consistency:** `rem` ensures uniform spacing and typography.

---

## difference between em and rem in CSS? 
- em units are relative to the parent’s font size, while rem units are relative to the root element (html).
- I prefer rem for consistent and scalable layouts, and em when I want elements to scale based on their parent’s size (e.g., buttons or icons).



## 🔹 1. `em` Example

```css
html {
  font-size: 16px;
}

div {
  font-size: 20px;
}

p {
  font-size: 1.5em; /* 1.5 × parent (20px) = 30px */
}
```
