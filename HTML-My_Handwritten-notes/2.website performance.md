## How do you optimize an HTML website for Performance?


1. **Minimize HTTP Requests**  
   Reduce the number of HTTP requests by combining CSS and JavaScript files.

2. **Use External CSS and JavaScript Files**  
   Store styles and scripts in external files for better caching and maintainability.

3. **Minify CSS, JavaScript, and HTML**  
   Remove unnecessary whitespace, comments, and formatting to reduce file sizes.

4. **Compress Images**  
   Use tools to compress images and choose appropriate formats.

5. **Leverage Browser Caching**  
   Set cache headers to allow resources to be stored locally, reducing the need for repeated requests.

6. **Load Critical Resources First**  
   Prioritize essential resources to improve perceived performance.

7. **Use Async and Await in JS**  
   Manage asynchronous code efficiently to avoid blocking execution.

8. **Implement Lazy Loading**  
   Improve initial load times by loading resources only when needed.
