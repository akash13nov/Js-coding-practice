# 📌 Closures

- A **closure** is the combination of a function bundled together 
(enclosed) with references to its surrounding state (the lexical 
environment).

**OR**
- A **closure** is an inner function that has access to the outer 
  (enclosed) function's variables.

**OR**

- A function along with its lexical scope bundled together forms a 
  closure.


### Lexical Scope
- Whenever an execution context is created, a lexical environment is 
created.
- Lexical environment is the local memory along with the lexical 
environment of its parent.
- Lexical refers to being in hierarchy or in sequence.
---


## 📌 Example 1
```
function outerFunction() {
    const outerVariable = "outer scope";

    function innerFunction() {
        console.log(outerVariable);
    }

    return innerFunction;
}

const closure = outerFunction();
closure(); // Output: outer scope
```

## 📌 Example 2
```javascript
function x() {
    var a = 7;
    function y() {
        console.log(a);
    }
    return y;
}

var z = x();
console.log(z); // function definition
z(); // 7
```
