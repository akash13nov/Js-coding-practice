### Javascript variable
- Variable is a temporary memory location which is used to store
  some data.
  **or**
- Variables stores the data which can be changed or used when we
  need.  

- There are three *keywords* to declare a variable.
- *keywords* are the predefined words in programming languages.

a) var 		var name = 10;
b) let 		let name = 10;
c) const	const pi = 3.14;


### Datatype in Javascript 
- there are two types of data

1) Primitive
2) Non-Primitive


### 1. Primitive Datatype
- Primitive data types goes in **stack memory** and it provide copy
  of that element.

- Primitive datatypes are :-
a) Number
b) Null
c) String
d) Boolean
e) Undefined
f) Bigint
g) Symbol  


### 2. Non Primitive Datatype
- Non Primitive data types goes in **Heap Memory** and it gives 
  reference value to assigned element.

- Non Primitive datatypes are :-
a) Array
b) Object
c) Function
