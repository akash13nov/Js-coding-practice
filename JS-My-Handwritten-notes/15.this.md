# 📌 `this` Keyword in JavaScript

The `this` keyword provides a way to access the current object or 
context.

---

## 📝 Notes

1. **Global Scope**  
   - In global scope, `this` always points to the **global object** 
   (`window` in browsers, `global` in Node.js).
   - Example:
     ```javascript
     console.log(this); // In browser: Window object
     ```

2. **Inside a Function**  
   - When a function is invoked in non-strict mode, `this` points to 
	 the **global object**.
   - In strict mode (`'use strict'`), `this` will be `undefined`.
   - Example:
     ```javascript
     function test() {
         console.log(this);
     }
     test(); // Global object in non-strict mode, undefined in strict 
	 mode
     ```

3. **Inside a Method**  
   - In a method, `this` always points to the **object** that invoked 
     the method.
   - Example:
     ```javascript
     const obj = {
         name: "Akash",
         greet: function() {
             console.log(this.name);
         }
     };
     obj.greet(); // "Akash"
     ```

4. **Using `call`, `apply`, `bind`**  
   - These methods can change the value of `this` to another object.
   - Example:
     ```javascript
     const person = { name: "John" };
     function greet() {
         console.log(this.name);
     }
     greet.call(person); // "John"
     ```
5. **Inside Arrow Functions**  
   - `this` inside an arrow function refers to the 
   **enclosing lexical context**, not the object that calls it.
   - Example:
     ```javascript
     const obj = {
         name: "Akash",
         arrowFunc: () => {
             console.log(this.name);
         }
     };
     obj.arrowFunc(); // undefined (refers to outer scope, not obj)
     ```

6. **`this` in the DOM**  
   - When used in a DOM event handler (normal function), `this` points
     to the **HTML element** on which it is being used.
   - Example:
     ```javascript
     document.querySelector("button").addEventListener("click", function() {
         console.log(this); // The button element
     });
     ```
