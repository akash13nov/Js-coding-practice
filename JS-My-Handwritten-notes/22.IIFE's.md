### 10. Immediately Invoked Function Expressions (IIFE)
or
### Instant function


An **Immediately Invoked Function Expression (IIFE)** is a JavaScript 
function that runs immediately after being defined.

---

#### **Key Points**
- **Purpose:**
  - Encapsulates code inside a **local scope** to avoid 
    **namespace collisions**.
  - Prevents variables and functions from leaking into the 
    **global scope**.

- **How It Works:**
  - When wrapped in parentheses `( ... )`, a function becomes a 
    **function expression**.
  - Adding another set of parentheses `()` immediately after invokes 
    it.
  - This is different from a **function declaration**, which is not 
    executed unless explicitly called.

- **Types of IIFE:**
  - **Anonymous IIFE** (most common)
  - **Named IIFE** (less common, useful for recursion or debugging)

---

#### **Syntax Examples**
```javascript
// Anonymous IIFE
(function() {
    console.log("Anonymous IIFE executed!");
})();

// Named IIFE
(function greet() {
    console.log("Named IIFE executed!");
})();
