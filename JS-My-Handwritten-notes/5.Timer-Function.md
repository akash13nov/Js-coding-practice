# ⏱ Timer Functions

A **timer function** is basically a function that has the ability to 
perform a task based on a time interval.

---

## 📌 Types of Timer Functions in JavaScript:
1. **setTimeout** → Executes a function **once** after a specified 
   delay.
2. **setInterval** → Executes a function **repeatedly** at specified 
   time intervals.

---

## 💻 Examples:

### 1. setInterval (Repeated Execution)
```javascript
function myFunction() {
    console.log("Hello");
}
const intervalId = setInterval(myFunction, 1000); // Runs every 1 second

// Stop after 5 seconds
setTimeout(() => {
    clearInterval(intervalId);
    console.log("Interval Stopped");
}, 5000);
```
---
### 2. setTimeout (One-time Execution)
```
function myFunction() {
    console.log("Hello");
}
const timeoutId = setTimeout(myFunction, 5000); // Runs once after 5 seconds

// Cancel the timeout before it executes
clearTimeout(timeoutId);
```
---

### 💡 Notes:

setTimeout() → One-time delay.

setInterval() → Repeats indefinitely until stopped using 
clearInterval().

clearTimeout(id) → Cancels a scheduled setTimeout.

clearInterval(id) → Stops a running setInterval.
