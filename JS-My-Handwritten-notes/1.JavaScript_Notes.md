
# 📜 What is JavaScript?

- JavaScript is a programming language that is used for converting 
static web pages to interactive and dynamic web pages.
- JavaScript is a lightweight, open-source, client-side scripting 
language developed by **NETSCAPE** in **1995**.
- In the beginning, it was known as **LiveScript**, but because it 
was generated by **Java**, NETSCAPE changed its name to **JavaScript**.

## 📜 Notes
- A **JavaScript engine** is a program present in a web browser that 
  executes JavaScript code.
---

## JavaScript Execution Nature
- JavaScript is a **synchronous** and **single-threaded** language.
- That means it can only do **one thing at a time** in a **specific 
  order**.
  
Everything in JavaScript happens inside an **execution context**.  

## 📌 Use of JavaScript
- JavaScript is used to create **dynamic web pages**.
- JavaScript is used to create richer (attractive web pages) 
  client-side interfaces of websites.
- JavaScript can be used for **security and validations**.

---

## ⚡ JavaScript Features

Features:
1. Case sensitive  
2. Dynamically typed  
3. Cross platform  
4. Interpreted  
5. Object-oriented scripting language  



# ⚙️ Ways to Run JavaScript

There are two ways to run JavaScript:  
1. **Interpreter**  
2. **Compiler**

---

## 📝 Interpreter
- Translates and reads the files line by line on the fly.
- Converts the JS code into bytecode.
- Interpreters are usually fast as they don't need to translate into 
another language.
