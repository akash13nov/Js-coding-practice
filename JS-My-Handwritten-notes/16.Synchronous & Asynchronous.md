# ⏳ Sync in JS

## Synchronous
- **Synchronous** means the code of instructions runs in a particular 
  sequence.
- Each instruction given in the program waits for the previous 
  instruction to complete its execution.

---

## Asynchronous
- **Asynchronous**: Due to asynchronous programming, sometimes the 
  instruction gets executed immediately and doesn’t block the flow due
  to some previous instructions, which causes a delay in the UI.

---

## Example of Asynchronous Programming

```javascript
console.log("one");
console.log("two");

setTimeout(() => {
    console.log("hello");
}, 3000);

console.log("three");
console.log("four");
