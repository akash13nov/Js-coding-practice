# 1 What is Function Statement?
A normal function that we create using Naming Convention & by this
we can do the Hoisting that is Function Statement.

```
function sayHello() {
    console.log("Function Statement");
}
```

# 2 What is Function Expression?
When we assign a function into a variable that is function Expression
& We cannot do Hoisting by this because it acts like variable.

```
var greet = function() {
    console.log("Function Expression");
};
```

# 3 What is Anonymous Function?
A function without a name is known as an Anonymous Function.
Anonymous functions are often used as values or arguments.

```
var greet = function() {
    console.log("Anonymous Function");
};
```

# 4 What is Named Function Expression?
A function with a name is known as Named Function Expression.

```
var a = function  xyz() {
	console.log("Names Function  Expression");
}
```

# 6 Difference between Parameters and Arguments

When we are creating a function, the values inside the parentheses 
`()` are called **parameters** (local variables for the function).

When we call the function and pass a value inside the parentheses 
`()`, that value is called an **argument**.

**Example:**
```javascript
function abc(param1, param2) {
    console.log(param1, param2);
}

abc(4, 5); // 4 and 5 are arguments
