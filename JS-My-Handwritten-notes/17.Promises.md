### Promises
========

1.  **Promises in JavaScript** are a way to handle asynchronous 
operations.
    
2.  A promise can be in one of three states:
    
    *   Pending
        
    *   Resolved
        
    *   Rejected
        
3.  A promise represents a value that may not be available yet but 
will be available at some point in the future.
    

const promise = new Promise((resolve, reject) => {

// Perform asynchronous operation (for example: API request)
// Call 'resolve' function when the operation succeeds
// Call 'reject' function when the operation encounters an error

});
