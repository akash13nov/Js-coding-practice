### 1. Call:

- Call is a function that helps you change the context of the invoking
  function.

In other words, it helps you replace of **this** inside a function 
with whatever value you want.  

```js
// call syntax:
Function.call(context, thisArg1, thisArg2, thisArg3);
```

### 2. Apply:

- apply is the exectly same is the call method the only different is
  it takes the second arguments as an array.
  
```js
// apply syntax:
Function.apply(context, [thisArg1, thisArg2, thisArg3]);
```
  
### 3. Bind:

- bind does not directly invokes the method but gives you the copy
  of that exectly same method which can be invoke later.  

```js
// bind syntax:
const newFunc = Function.bind(context, thisArg1, thisArg2, thisArg3);
```
  
```
const person1 = {
		FName : "Vijay",
		LName : "Mahta",
		FullName : function (hometown,country) {
			return this.FName + " " + this.LName+ " " +hometown+" "
			+ country
		}
}
const person2 = {
	FName : "Aman",
	LName : "Jain"
}

console.log(person1.FullName.call(person2,"Pune","India"));

// Apply
console.log(person1.FullName.apply(person2,["Pune","India"]));

// Bind
const result = person1.FullName.bind(person2,"Pune","India")
console.log(result)
console.log(result());
```  

### short notes

a) Call invokes the function and allows you to pass in arguments 
one by one.
b) Apply invokes the function and allows you to pass in arguments as 
an array.
c) Bind returns a new function, allowing you to pass in this array 
and any number of arguments.
