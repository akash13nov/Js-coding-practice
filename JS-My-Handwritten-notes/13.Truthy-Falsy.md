# Truthy and Falsy Values


## Falsy Values
- A falsy value is a value that translates to `false` when evaluated 
  in a Boolean context.  
- JavaScript treats the following values as falsy:  
`false`, `0`, `-0`, `""` (empty string), `null`, `undefined`, `NaN`  

Example:  
```javascript
if (0) {
  console.log("This won't execute");
}
```

## Truthy Values

- A truthy value is a value that translates to true when evaluated in 
a Boolean context.

Examples: `{}, [], "0", "false", " " (space), 23`
