#### Deep Copy



##### Deep Copy means everything will be copied. Shallow copy means something external is copied but not the whole thing copied.
---

Example : 
Deep copy using JSON.parse() and JSON.stringify()
```js
const persone = {
  name : 'Happy',
  age : 30,
  address : {
    city: 'Delhi',
    country : 'India'
  }
}


const deepCopy = JSON.parse(JSON.stringify(person));

deepCopy.address.city = "Mumbai";

console.log(person.address.city); // output : "Delhi"
console.log(deepCopy.address.city); // output : "Mumbai"
```
#### Shallow Copy

Shallow copy means something external is copied but not the whole thing copied.

Example : 
// Shallow copy using Object.assign()
```js
const persone = {
  name : 'Happy',
  age : 30,
  address : {
    city: 'Delhi',
    country : 'India'
  }
}


const shallowCopy = Object.assign({}, person);

shallowCopy.address.city = "Mumbai";

console.log(person.address.city); // output : "Delhi"
console.log(shallowCopy.address.city); // output : "Delhi"
```

question3();

<strong><u>Note:</u></strong> If a object is nested, Object.assign or Spread operator won't make a deep copy for nested object
