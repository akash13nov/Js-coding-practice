**splice() vs slice()**

# ✂️ splice() vs slice() in JavaScript

### **1. splice() Method**
- Works **only on arrays**.
- Modifies the **original array** by adding, removing, or replacing 
  elements at specified positions.
- Returns the removed elements.
- **Mutates** the original array.

### Example:
```javascript
let arr = [0, 1, 2, 3, 4];
let removed = arr.splice(1, 2, "x", "y"); // remove 2 elements from 
index 1 and insert "x", "y"

console.log(removed); // [1, 2]
console.log(arr); // [0, "x", "y", 3, 4]
```

### **2. slice() Method**

-Works on both arrays and strings.
-Does not modify the original array or string.
-Returns a shallow copy of a portion of the array or string.

### Example:

**Example (Array):** 
```
let arr = [0, 1, 2, 3, 4];
let part = arr.slice(1, 3);

console.log(part); // [1, 2]
console.log(arr);  // [0, 1, 2, 3, 4]

```

**Example (String): 
```
let str = "javascript";
let partStr = str.slice(4, 6);

console.log(partStr); // "sc"
console.log(str);     // "javascript"
```


### 📌 Key Difference:

| Feature              | splice()           | slice()          |
| -------------------- | ------------------ | ---------------- |
| **Mutates Original** | ✅ Yes              | ❌ No             |
| **Return Type**      | Removed elements   | Copied portion   |
| **Works on**         | Arrays only        | Arrays & Strings |
| **Usage**            | Add/Remove/Replace | Copy portion     |




#### 2. When to use reduce(), map(), foreach() and filter() in JavaScript?

##### reduce:

- When you want to accumulate values from an array into a single value
  (summing up values, avg calculation, concatenating strings)

- takes accumulator and current as arguments and returns the 
  accumulated result.

- You can also provide an initial value for the accumulator as the 
  second argument.

```js
const numbers = [1, 2, 3, 4, 5];

const sum = numbers.reduce((accumulator, current) => accumulator + 
current, 0);
console.log(sum); // 15
```

##### map:

- When you want to transform each and every element of an array into 
  a new value based on the specified logic.

- It creates a new array.

```js
const numbers = [1, 2, 3, 4];

const doubledNums = numbers.map((item) => item * 2);
console.log(doubledNums); //[2, 4, 6, 8]
```

##### foreach():

- When you want to iterate through an array and perform an action on 
  each element, but you don't want to create a new array or accumulate a single value.

```js
const nums = [1, 2, 3, 4];

nums.forEach((num) => console.log(num * 2)); //[2, 4, 6, 8]
```

- Modifies the original array

##### filter:

- When you want to create a new array containing elements that pass 
  the condition (returns the new array that includes only elements that satisfied the condition)

```js
const numbers = [1, 2, 3, 4, 5, 6];

const filteredArray = numbers.filter((item) => item % 2 === 0);

console.log(filteredArray); // [2, 4, 6]
```
